# دليل تشغيل بوت CHAOS CODEX 🚀

لقد قمت بتجهيز البوت لك بالكامل، وبما أن هناك قيوداً تقنية على الاتصال بتيليجرام من داخل بيئة الفحص، يمكنك تشغيله على جهازك (كمبيوتر أو خادم) في دقيقة واحدة باتباع الخطوات التالية:

## 1. المتطلبات
تأكد من تثبيت **Python 3.10** أو أحدث على جهازك.

## 2. التثبيت
افتح "موجّه الأوامر" (Terminal / CMD) في مجلد البوت وقم بتشغيل الأمر التالي لتثبيت المكتبات اللازمة:
```bash
pip install -r requirements.txt
```

## 3. التشغيل
قم بتشغيل البوت باستخدام الأمر:
```bash
python main.py
```

## 4. معلومات هامة
- **التوكن مفعل:** البوت يستخدم التوكن الخاص بك المدمج في الكود.
- **واجهة الويب:** يمكنك أيضاً تشغيل واجهة الويب عبر `python app.py` والوصول إليها عبر `http://localhost:2590`.
- **الحسابات:** البوت يستخدم الحساب الموجود في `accs.txt` لجلب البيانات.

**البوت جاهز تماماً للعمل بمجرد تشغيله على جهاز يمتلك اتصالاً مفتوحاً بتيليجرام.**
